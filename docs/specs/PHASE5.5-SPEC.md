# 階段 5.5 — 員工自助門戶（1.5 週）

**Kairo Task**: `mr7meczx9lzzx` | **分支**: `phase5.5/employee-portal`

## 目標

員工能看到自己的班表、考勤、假期，並發起各種申請。

## 功能範圍

- **我的班表**：週/月視圖，即時推送更新
- **我的考勤**：打卡歷史，原始 vs 補登一目瞭然
- **我的假期**：餘額 + 申請狀態
- **發起申請**：補打卡/請假/換更（走審批 + audit_log）
- **月度摘要**：工時/加班/請假（不顯示薪資）
- **通知**：申請結果、班表變動推送

## 驗收標準
- [ ] 員工只看到自己的資料
- [ ] 無法直接編輯考勤/班表
- [ ] 發起申請 → 審批 → 通知，全程留痕
- [ ] `npm run build` 通過
